2. the seven categories of named entities in Haskell?

    term-level
    variables
    data constructors
    type variables
    type constructors
    typeclasses
    modules

3. a typeclass in Haskell?

is an interface.

4. data constructor in Haskell?

is a means to create values that inhabit a given type.

5. type constructor in Haskell?

constructs typeclass; is not value and can only be used in type signatures.

6. term-level in Haskell?

is where values live and is the code that executes at runtime.

7. type-level in Haskell?

is used during static-analysis & verification of program, i.e. at compile-time. Have type-variables, type-constructors, typeclasses.

8. module in Haskell?

is used to organize code into coherent groupings across different files.

9. id and class selectors in CSS?

use # for id; use . for class
